# Piano Urbano

Mapa interativo com pianos públicos espalhados por São Paulo. Veja detalhes de cada instrumento e trace rota até ele pelo Google Maps ou Waze, ou cadastre um novo piano que você encontrou pela cidade.

## Stack

Next.js (App Router) · React · TypeScript · Tailwind CSS · Leaflet/react-leaflet · shadcn/ui

## Rodando localmente

```bash
pnpm install
pnpm dev
```

Abra [http://localhost:3000](http://localhost:3000).
